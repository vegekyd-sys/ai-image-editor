## captions（创意排版）— 2个

把照片变成一个有设计感的视觉作品。文字不是加上去的标签——它是画面叙事的一部分。

---

## ⚠️ 前置检查：画面里有没有已存在的文字？

先扫一遍画面，判断是否有文字（招牌、标语、水印、logo、路牌等）：

- **没有文字** → 正常进入四问自检
- **有文字，且是画面故事的一部分**（如餐厅招牌、路牌、书名）→ 保留，新 caption 要与它和谐共存，不要遮挡它
- **有文字，但是杂乱/无关/破坏观感**（如水印、无关广告牌）→ 在 editPrompt 里先清除这些文字，再加入新 caption
- **有文字，且可以被创意替换**（如店名可以改成有趣的版本，路牌可以改成诗意的词）→ 可以选择用新文案替换原有文字，作为一种创意手法

---

## ⚡ 先做这一步：找到这张照片的「隐藏角度」

在进入自检之前，必须先问自己：

**这张照片表面在说 A，但如果换一个角度看，它其实可以是 B。**

找反差、找隐藏故事、找意外解读：
- 游客在博物馆看雕像 → 换角度：雕像在展览人类观众（"EGYPT AWAKENED"）
- 朋友聚餐吃火锅 → 换角度：热气腾腾的江湖义气，或者"今晚，热量不计"
- 情侣在大桥前拍照 → 换角度：这不是旅行照，这是一部公路爱情片的海报
- 独自旅行的女生 → 换角度：她不是游客，她是这座城市今天的主角
- 孩子第一次见到海 → 换角度：宇宙级别的第一次接触

找不到隐藏角度？再看一遍照片，想：**如果这是一部电影/一本书/一个展览，它的名字是什么？为什么？**

## 五问自检（找到隐藏角度后，逐一回答）

**Q1：这张照片在说什么故事？**
用一句话说清楚：画面里正在发生什么，情绪是什么，谁是主角。

**Q2：如果要给这张照片加文字，这段文字扮演什么角色？**
从以下四种角色里选最适合的一个——必须说出理由：
- **旁白者**：画面之外的声音，说出画面没说完的。适合情绪强的照片。
- **记录者**：时间+地点的印记。⚠️ 记录者必须有具体地点，纯时间戳=3分。可加场景相关小图标（小船/山峰/咖啡杯）。
- **人物内心**：主角此刻的想法——不是气泡（气泡很呆），是融入画面的轻声细语。
- **标题设计师**：把照片变成海报/杂志大片，用"隐藏角度"里找到的概念做片名/标题。

  ⚠️ **美学参考（必须读完）**：

  **参考1 — 极简数字**：4个人走在街上，画面正中只有一个大号圆体白字"**2021**"，没有任何 tagline 或说明。就这一个词/数字，照片本身不动。极简即极强。

  **参考2 — MSCHF 风格**：全大写、mono/工业字体、高对比黑白、一句话说死不解释。冷静、前卫、provocative。

  **参考3 — Dazed 杂志封面**：顶部超大 "DAZED" 红色衬线 masthead，中间手写白字 "Mk.gee" 随意放在有空间的地方——不居中、不在设计区域，直接写在照片上。照片没有被"装饰"，而是被"命名"。只有3个文字元素。手写字可以有颜色，可以随意，但位置有构图逻辑。

  **禁止**：tagline + credits 的好莱坞结构（"A FILM ABOUT FRIENDSHIP" 下面跟一排演员名——这是2010年的 cheap）。多行文字堆叠 = 设计失败。

  **正确公式**：**1-3个词/1个数字/1句话，放对位置，字够大，照片不动**。

  ---

  **海报/杂志设计 3 问（写 editPrompt 之前必须全部过关）：**

  **Q海报1：这个设计有没有 ONE bold choice——一个让人说"这挺大胆"的决定？**
  可以是：
  - 一个意外的颜色（DAZED 用红色 masthead；照片是暖橙色就用冰蓝字）
  - 一个夸张的尺寸（字母高度占画面高度 20%+，敢于大）
  - 一个非中心的位置（Dazed 手写字放中部左侧，不在任何"安全区"）
  想不出 → 重新想，不能没有。

  **Q海报2：设计里有几个文字元素？超过 2 个，删到只剩 1-2 个。**
  主标题 + 可选地点/日期 = 最多 2 个。无条件删掉：
  - tagline（解释性副标题，如 "A film about friendship"）
  - credits（演员/导演行）
  - 任何"补充说明"性质的文字
  超过 2 个 = 廉价感触发器。

  **Q海报3：把这个设计贴到 Instagram，2025 年的用户会停下来吗？**
  - ✅ A24 海报风（文字作为主视觉，非对称，非好莱坞配色）
  - ✅ Supreme 图形（一句话，一个颜色块，不需要解释）
  - ✅ Dazed 封面（大 masthead + 一个有机手写元素，就这样）
  - ❌ "2010年 DVD 封面感" → 重来
  - ❌ "旅游景点宣传图感" → 重来

**Q3：文字放在哪里，才感觉像是属于这张照片的？**
必须有构图理由（视线方向 / 天空留白 / 配合场景元素）。

**Q4：如果把这段文字去掉，画面会损失什么？**
- ✅ 少了一层情绪或叙事完整感 → 有价值
- ❌ 去掉后照片更干净 → 这段文字是多余的，重新想

**Q5：这个 caption 有没有让人「惊喜」的瞬间？**
惊喜 = 看完后有反应：会心一笑 / 意外的感动 / 出人意料但又非常贴 / 黑色幽默。

具体检验方法：
- 把这段文案念给没见过这张照片的人听，他们会好奇"这是哪张照片"吗？→ 好奇=有惊喜
- 这个片名/标题，10个摄影师里有几个会想到？→ 1-2个才想到=高分，10个都会想到=换
- 有没有一个词或一句话让人"没想到还能这么说"？→ 有=好

- ❌ 文案感觉"正确"但没有任何意外 → 太平，用「隐藏角度」重新出发
- ❌ "设计感强但没有灵魂" → 概念失败，回到最开始找隐藏角度

## 执行标准

回答完四个问题后，写 editPrompt 时必须包含两部分：

### Part 1：照片本身如何变化来迎合这个设计
文字和照片是一个整体作品，照片要主动配合文字的风格。
⚠️ 关键约束：**照片调整幅度要克制**——增强氛围感可以，但构图/人物/主体场景不能大幅改变，看完还要能一眼认出是同一张照片。

- 做**电影海报** → 增强对比度和光影戏剧感，顶部/底部加暗化渐变为文字让位，保持场景和人物不变
- 做**旅行杂志大片** → 轻微调色（清透/胶片感），不裁切主体，在留白区域加文字
- 做**电影旁白字幕** → 加黑边（letterbox），轻微电影色调，字幕在黑边内
- 做**胶片记录** → 轻微颗粒感和暖色调，主体完全保留
- 做**人物内心** → 局部柔化，背景轻微虚化，整体构图不变

### Part 2：文字设计
1. **文案内容**：根据照片内容创作具体的文字（Gemini 自己发挥，要有创意和画面感）。如果画面或 metadata 里有**具体地点名**（博物馆、城市、景点），必须用进文案——具体地点比泛化描述好 10 倍
2. **视觉处理**：文字怎么融入画面——渐变遮罩 / 黑边 / 底部留白区 / 胶片边框 / 海报构图
3. **字体方向**：与角色和情绪匹配
   - 旁白者 → 细白 sans-serif，克制，像电影字幕
   - 记录者 → 橙色数字胶片体 / 手写小字；⚠️ 胶片时间戳**必须包含地点**（没有地点的时间戳 = 3分）；语言统一不要中英混排；可加场景相关小图标增加个性

     🚫 **METADATA 死规则（违反 = 自动失败）**：
     - **时间**：只能用 `[照片元数据]` 里明确给出的时间。没有 metadata？→ 省略年份，只写月份或季节（"秋"/"OCT"），绝不猜年份
     - **地点**：只能用 metadata 里的地点 OR 画面里能直接看见的标志（招牌/logo/地标）。看不出来？→ 只写大范围（"中国南方"而不是猜"桂林"），或完全省略
     - **宁可什么都不写，也不能写错** — 用户看到自己从没去过的地点会立刻 0 分
   - 人物内心 → 轻盈手写，融入而非突出
   - 标题设计师 →
     - **字号用占比描述**：letters tall enough to span 18-25% of frame height（禁止说 "big" 或 "large"，没有意义）
     - **颜色要有理由**，选"意外对"的颜色：暖橙/金色画面 → 冷白/冰蓝；中性灰街景 → 大红；蓝色背景 → 黄/橙。禁止：默认白色+渐变遮罩（这是所有廉价海报的通用默认值）
     - **editPrompt 里必须包含一个风格词**（选一个）："A24 arthouse poster" / "Dazed editorial cover" / "Supreme bold graphic" / "brutalist minimal" / "oversized single-word treatment"
4. **位置逻辑**：说清楚为什么放这里（来自 Q3 的答案）
5. **人脸保真**：Preserve each person's face, bone structure, and skin exactly

---

## 品质标准

- 两个 tip 必须使用不同的角色（Q2 里选的不能一样）
- 文字渲染必须 photorealistic
- 胶片时间戳：橙色数字、右下角、复古数字字体，这样才真实
- 电影旁白：上下黑边必须对称，字幕在黑边内，不覆盖画面主体
- 海报：标题字够大（至少占画面宽度 35%），天空/背景区域加遮罩让字可读
- 禁止：对话气泡（很呆）、随意贴在照片上没有构图逻辑的文字
